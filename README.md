## 📖 About MOLE

**MOLE (Marketplace of Legends)** is a web-based marketplace built specifically for the action figure collector community. The platform addresses the lack of a dedicated, trustworthy space for collectors to buy, sell, and interact — offering a secure transaction system, social features, and a gamified loyalty experience.

This project was developed as part of the **Project-Based Learning (PBL)** course at **Politeknik Negeri Batam**, applying the **AGILE process model** throughout development.

---

## ✨ Key Features

### 🛒 Marketplace
- Curated action figure listings with detailed product information
- Secure checkout flow with complete purchase history
- Wishlist management for tracking desired items

### 🤝 Community
- Interactive content feed — users can like, comment, and share posts
- Collector profiles with personal showcase pages

### 🎖️ Gamification & Loyalty
- **EXP-based leaderboard** that rewards active buyers with collector titles:
  - 🥉 Bulk Buyer
  - 🥈 Loyal Hunter
  - 🥇 Premium Collector

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/Verrelian/jualfigure.git
cd jualfigure

# Install dependencies
composer install
npm install

# Set up environment
cp .env.example .env
php artisan key:generate

# Run database migrations
php artisan migrate --seed

# Start the development server
php artisan serve
```
